Why this boilerplate?

At Bitnami we use middleman to create static sites for our products. From simple sites to complex static applications, middleman fits all our requirements.

For this reason, we've created this boilerplate with the basic configuration of our static site projects.

How does it work?

This boilerplate follows the basic structure of middleman projects. All the files related to the site generation are place in the source folder. You only need to fork the project and start using it. It also includes a docker-compose file, so you don't need to install any extra library.

Get started

Let's start with your next awesome project. Just fork this repository and start modifying the content. You have several useful examples in the repository. Don't forget to check them if you need some inspiration ;)
